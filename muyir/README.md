# mûyir 
-----
- pronounciation: `[mÿjɾ]`

- language family:
`muy-gulpean` > `muyiric` > `central muyiric`

- early forms
`proto muy-gulpe` > `old muyir` > `middle muyir` > `late muyir` > `modern muyir`

- ACRONYMS: `PMG` `OM` `MM` `LM` `DM`

- writing system: muyiric handscript

- grammar `SOV`
